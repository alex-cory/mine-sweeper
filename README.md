Mine Sweeper
============

#### Bult using `styled-components`, `mobx`, and `react`

Play with it [here](https://alex-cory.github.io/mine-sweeper/)!

Right click to flag or unflag a cell! :)

#### Installation & Running
- `yarn` or `npm install`
- `yarn start` or `npm start`
